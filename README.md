# 📧 Newsletter Subscription Card (Cadastro de E-mail)

> Um componente de captura de leads (newsletter) elegante e responsivo, desenvolvido com HTML5 e CSS3, focado em alta conversão e organização semântica.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir um **Card de Cadastro de E-mail** robusto e visualmente atraente para ser utilizado em landing pages ou blogs. 

O componente foca na simplicidade e na confiança do usuário, apresentando uma chamada clara para ação e garantias de privacidade.

## ✨ Destaques e Aprendizados Técnicos

* **Arquitetura Modular:** Divisão clara do componente em contêineres (`wrapper`, `top`, `bottom`), permitindo fácil manutenção e escalabilidade do layout.
* **Formulários Estruturados:** Implementação de uma estrutura de formulário com campos de entrada (`input type="text"`) e botões de envio (`submit`), preparada para integração com back-end.
* **Feedback e UX:** Inclusão de uma área de rodapé informativa para reduzir o atrito na conversão, garantindo ao usuário que não será enviado spam.
* **Integração de Bibliotecas Externas:** Uso eficiente da biblioteca **Font Awesome (v5.15.2)** para fornecer suporte visual e icônico ao componente.
* **Design Centrado no Usuário:** Foco na clareza das informações (`info`) e no posicionamento estratégico dos elementos de interface para guiar o olhar do visitante.
* **Separação de Responsabilidades:** Código organizado seguindo padrões profissionais, com a estrutura no `index.html` e a identidade visual em uma folha de estilos externa (`css/style.css`).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e lógica de formulário.
* **CSS3:** Estilização de layout, gerenciamento de cores e design responsivo.
* **Font Awesome (v5.15.2):** Iconografia vetorial para suporte visual.
